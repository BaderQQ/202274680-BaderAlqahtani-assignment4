# AI Usage Report

## 1. Tools Used and Use Cases

### ChatGPT
I used ChatGPT to support:

- planning the website structure
- improving content wording so the site feels like a real personal website
- reviewing JavaScript logic for filtering, saved preferences, validation, and API handling
- improving documentation clarity
- preparing presentation slides 

### GitHub Copilot
I used GitHub Copilot to support:

- small code completions while editing HTML, CSS, and JavaScript
- repeated syntax patterns
- minor refactoring suggestions

---

## 2. Benefits

AI tools helped me work faster and think through different design and implementation options. They were especially helpful for improving wording, checking JavaScript logic, organizing documentation.

---

## 3. Challenges

AI suggestions were not always ready to use directly. Some suggestions were too formal, too generic, or too focused on showing technical requirements instead of creating a natural personal website. I had to review and rewrite content so the final result matched my own voice and long-term purpose.

---

## 4. Learning Outcomes

Through this process, I improved my understanding of:

- structuring a complete static website
- managing browser state with `localStorage`
- using JavaScript to filter and render data
- handling API loading and error states
- validating forms and escaping user-generated content
- writing documentation that is useful for future maintenance

---

## 5. Responsible Use and Modifications

AI was used as a support tool, not as a replacement for understanding the work. I reviewed the generated suggestions, modified wording and code, removed unnecessary complexity, tested the features, and made sure the final website represented my own goals and style.
